Gegeben sind drei Zahlen, welche mit einem \emph{print()}-Befehl ausgegeben werden sollen, sodass sie untereinander stehen und linksbündig geschrieben sind.

Verzichten Sie auf den Seperator-Parameter und Formatierungsanweisungen wie "'<"' für die Linksbündigkeit. Nutzen Sie hierfür die minimale Länge mit Wert 10.

\begin{lstlisting}[language=Python, label=minLength:lst:minLength]
# Ausgabe mittels minimaler Länge

a = 1234567890
b = 12345
c = 5

print(...)

# Ausgabe:
# 1234567890
#      12345
#          5
\end{lstlisting}
