Schreiben Sie eine Funktion, die die Punkte der ausgewählten Form an das Canvas-Widget
übergibt und diese zeichnet. Nutzen Sie hierzu die Methode \lstinline$create_polygon()$
des Canvas-Widget. \lstinline$create_polygon()$ erwartet als erstes Argument eine Liste mit
Koordinaten. Übergeben Sie als zweites Argument eine Farbe mit \lstinline$fill=$.
Beispiel: \lstinline$fill="yellow"$.
Übergeben Sie die Funktion anschließend dem Zeichnen-Button mithilfe des \lstinline$command$-Attributs von
Button.
