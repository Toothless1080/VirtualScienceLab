Schreiben Sie eine kleine Klasse, deren instanziiertes Objekt eine geometrische Figur
verkörpert und legen Sie anschließend statisch zwei geometrische Figuren an.

In unserem Fall genügt es, der Klasse die Attribute Namen, Points und Color zu geben,
um eine geometrische Figur zu verkörpern. Points besteht dabei aus einer Liste mit
X- und Y-Koordinaten. Um ein Quadrat anzulegen, wird also eine Liste mit Vier punkten
benötigt, wobei jeder Punkt aus zwei Werten besteht. Beispiel: [0, 0, 0, 100, 100, 100, 100, 0]
erzeugt ein Quadrat mit den Punkten (0/0), (0/100), (100/100) und (100,0).
