Schreiben Sie ein Programm, das einen Thread erzeugt, der die Zahlen 1 bis 10 ausgibt, und sich
dann beendet. Zwischen den Ausgaben soll der Thread eine Sekunde warten.

\tip{
Um einen Thread warten zu lassen, können Sie die \lstinline$sleep()$-Methode verwenden. Diese Methode 
kommt aus dem \lstinline$time$-Modul und nimmt die Zeit in Sekunden, die zu schlafen ist, als
Parameter entgegen.
}
