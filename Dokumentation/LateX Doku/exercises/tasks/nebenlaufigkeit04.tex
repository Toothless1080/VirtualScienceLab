Erweitern Sie die \lstinline$increment()$-Methode der \lstinline$Counter$-Klasse um die Möglichkeit,
einen Wert anzugeben, um den inkrementiert werden soll.
Ist der angegebene Wert größer 1, so soll die \lstinline$increment()$-Methode sich rekursiv selbst 
aufrufen.
Der Wert des \lstinline$Counters$ soll also immer nur direkt um 1 erhöht werden.
Ergänzen Sie in der \lstinline$IncrementerThread$-Klasse den neuen Parameter beim Aufruf von
\lstinline$increment()$.
Was fällt Ihnen auf, wenn Sie das Programm ausführen?
