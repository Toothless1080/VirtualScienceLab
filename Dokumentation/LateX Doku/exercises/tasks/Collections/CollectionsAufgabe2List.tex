\begin{enumerate}
	\item Erzeugen Sie eine List und befüllen Sie diese mit den Elementen 1 bis 10 in beliebiger Reihenfolge.
	\item Geben Sie die Summe aller Elemente der List auf der Konsole aus.
	\item Geben Sie den größten Zahlwert der List auf der Konsole aus.
	\item Erzeugen Sie eine weitere List mit den Elementen 0 und 1. Schreiben Sie anschließend ein Programm, das die List mit den ersten zehn Fibonacci-Zahlen befüllt.
	
	\tip{
		In der Fibonacci-Zahlenfolge ergibt die Summe zweier aufeinander folgenden, natürlichen Zahlen den nächsten Zahlwert.
	}	
\end{enumerate}
