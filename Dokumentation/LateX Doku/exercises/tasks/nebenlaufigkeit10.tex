\label{nebenlaufigkeit10}
Im Folgenden soll die Klasse \lstinline$Queue$ in einem Beispiel verwendet werden.
\begin{enumerate}
\item Erstellen Sie einen \lstinline$Queue$ und befüllen Sie ihn mit den Zahlen von 1 bis 20.

\item Erzeugen Sie drei Dämon-Threads, welche Elemente aus dem erzeugten \lstinline$Queue$ entnehmen
und das Quadrat der Elemente ausgeben.

\item Nachdem die Threads gestartet wurden, soll der Main-Thread darauf warten, dass alle Elemente 
vollständig abgearbeitet werden, bevor er sich beendet.
\end{enumerate}
