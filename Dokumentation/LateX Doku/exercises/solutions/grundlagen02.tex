Es muss eine neue .py Datei mit einem beliebigen Namen erstellt werden.
In dieser Datei muss das Kommando \lstinline$print("Hallo Python-World!")$ stehen.
Ein Beispiel einer solchen Scriptdatei finden sie im Ordner für Quelltexte unter dem namen \textit{grundlagen02.py}
Im Anschluss muss die Datei über den Befehl \lstinline$Python[Versionsnummer] [Scriptname]$ ausgeführt werden.
Es sollte im Anschluss der eingegebene Text auf der Konsole erscheinen.
