Bei einer Eingabe auf der Konsole arbeitet Python mit dem Typ \lstinline$str$.
Aus diesem Grund ist eine Konvertierung zu einer Zahl an dieser Stelle sinnvoll.
Anschlieﬂend wird die Eingabe quadriert und im fehlerfreien Fall auch die \lstinline$else$-Anweisung durchlaufen.
Kommt es hierbei zu einem Eingabefehler z. B. bei der Eingabe des Literals \glqq{}a\grqq{}, tritt eine Ausnahme auf.
