Importieren Sie Tkinter mit der Anweisung \lstinline$from tkinter import*$.
Erzeugen Sie anschließend ein Label-Widget mit der Anweisung \newline
\lstinline$label = Label(root, text=’Hallo Python-World!’)$ und fügen Sie es dem
angegeben Wurzel-Elemtent 'root' mit einem der drei Geometry-Manager hinzu.
\lstinline$label.grid()$, \lstinline$label.pack()$ oder \lstinline$label.place()$.

Ein Beispiel der Anwendung finden sie im Ordner für Quelltexte unter dem namen \textit{GUIHelloWorld.py}.
