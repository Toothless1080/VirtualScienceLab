In diesem Kapitel gehen wir auf die Nutzung von Funktionen in Python ein.
Eine Funktion bildet einen Code-Block ab, also eine Sequenz von Befehlen, die eine bestimmte \emph{Funktion} erfüllt. 


Dieser Code-Block wird mit dem Schlüsselwort \lstinline$def$ gestartet, gefolgt vom Namen der Funktion, anschließend von Klammern (welche Input-Parameter beinhalten können). Zum Schluss der Funktionsdefinition folgt noch ein Doppelpunkt, nach diesem kommt die Befehlssequenz.
Soll die Funktion Werte zurückliefern, dann steht am Ende der Sequenz das \lstinline$return$-Schlüsselwort.

\begin{lstlisting}[language=Python, label=einleitungFunktionen:lst:funcDef]
# Definition einer Funktion in Python

def funktionsname (parameter):
    ...
    return rueckgabewert
\end{lstlisting}

\warning{Eine Funktion liefert immer einen Wert zurück. Wird keiner angegeben, so wird der Wert \lstinline$None$ als Rückgabewert festgelegt.}
